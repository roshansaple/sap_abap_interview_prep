You can say:

“ECC uses traditional disk-based databases like Oracle or SQL Server, where data is stored in rows and most processing happens in the application server.
SAP HANA is an in-memory database where data is stored in RAM using column-based storage. Because of this, data retrieval and analytics are much faster.
In HANA we also follow the code pushdown approach using CDS views and AMDP, which moves processing to the database and improves performance.”
